Politica de Privacidade

Ultima atualizacao: 20/02/2026

1. Dados coletados

Coletamos dados de autenticacao da conta (como e-mail) e dados de uso da ferramenta (cofres, metas e registros de aporte).

2. Finalidade do tratamento

Os dados sao utilizados exclusivamente para permitir login, funcionamento da plataforma e sincronizacao dos cofres do proprio usuario.

3. Nao exposicao de valores

Nenhum valor guardado pelo usuario na conta e exposto publicamente pela plataforma.

Os dados de cofres ficam vinculados a conta do proprio usuario e nao sao disponibilizados para terceiros na interface publica.

4. Protecao contra vazamento

Aplicamos controles de acesso, regras de seguranca e boas praticas de protecao para impedir vazamento de dados.

O Banco Ocioso nao comercializa nem divulga os dados pessoais e financeiros registrados na plataforma.

5. Direitos do usuario

O usuario pode solicitar atualizacao ou exclusao da conta e dos dados associados, conforme as regras aplicaveis.

6. Contato

Para assuntos de privacidade, utilize o canal oficial do projeto Banco Ocioso.
