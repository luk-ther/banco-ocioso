Política de Privacidade

Última atualização: 20/02/2026

1. Dados coletados

Coletamos dados de autenticação da conta (como e-mail) e dados de uso da ferramenta (cofres, metas e registros de aporte).

2. Finalidade do tratamento

Os dados são utilizados exclusivamente para permitir login, funcionamento da plataforma e sincronização dos cofres do próprio usuário.

3. Não exposição de valores

Nenhum valor guardado pelo usuário na conta é exposto publicamente pela plataforma.

Os dados de cofres ficam vinculados à conta do próprio usuário e não são disponibilizados para terceiros na interface pública.

4. Proteção contra vazamento

Aplicamos controles de acesso, regras de segurança e boas práticas de proteção para impedir vazamento de dados.

O Banco Ocioso não comercializa nem divulga os dados pessoais e financeiros registrados na plataforma.

5. Direitos do usuário

O usuário pode solicitar atualização ou exclusão da conta e dos dados associados, conforme as regras aplicáveis.

6. Contato

Para assuntos de privacidade, utilize o canal oficial do projeto Banco Ocioso.
